Instructions

Percy is a PC Agent controller for RTX PCs. Follow these steps to get started:

  1. Click the "Download Percy" button above to download Percy.exe.
  2. Save the file to a folder (e.g., Desktop).
  3. Double-click Percy.exe to launch the app. If you see a "Windows protected your PC" warning, click "More info" and "Run anyway".
  4. Use Percy to know more about your computer! (GPU, CPU, RAM, etc.).

Requirements: Windows, RTX GPU, internet connection, PC.
